# Playwright Python Automation Framework

A scalable UI automation framework built with:

- Python
- Pytest
- Playwright
- Page Object Model (POM)

## Architecture

- Config-driven base URL
- Reusable fixtures
- Page Object abstraction
- Modular structure

## Tech Stack

- Playwright
- Pytest
- JSON Config
- Headless browser execution

## Future Enhancements

- CI/CD Integration (GitHub Actions)
- Parallel Execution
- HTML Reporting
- Logging Integration

## How to Run

```bash
pip install -r requirements.txt
playwright install
pytest
